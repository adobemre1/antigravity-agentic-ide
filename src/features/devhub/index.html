Antigravity API

Programmatically orchestrate AI agents, manage swarms, and integrate Gemini 3 Pro reasoning into your own applications.

Authentication

The Antigravity API uses API keys to authenticate requests. You can view and manage your API keys in the Dashboard.

curl https://api.antigravity.dev/v1/agents \ -H "Authorization: Bearer ag_sk_test_..."

Create an Agent

Spins up a new autonomous agent container with specified capabilities.

POST /v1/agents
import antigravity client = antigravity.Client(api_key="...") agent = client.agents.create( model="gemini-3-pro", role="frontend-engineer", tools=["browser", "terminal"] ) print(agent.id) # ag_8x92m...
> 200 OK: Agent created successfully via Gemini 3 Pro.
