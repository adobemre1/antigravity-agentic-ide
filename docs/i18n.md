# Internationalization (i18n)

The project supports multiple languages (currently Turkish `tr` and English `en`) using `i18next`.

## Structure

Translation files are located in `src/locales/` (or embedded in `src/i18n.ts` for simple apps).

- **Current Setup:** Translations are defined in `src/data/` or `src/i18n.ts`.

## Adding a New Language

1.  Create a new translation resource object (e.g., `translationDe`).
2.  Register it in `src/i18n.ts`:
    ```typescript
    resources: {
      tr: { translation: translationTr },
      en: { translation: translationEn },
      de: { translation: translationDe }, // Add this
    }
    ```
3.  Update the `LanguageSwitcher` component to include the new option.

## Using Translations

Use the `useTranslation` hook in components:

```tsx
import { useTranslation } from 'react-i18next';

export default function MyComponent() {
  const { t } = useTranslation();
  
  return <h1>{t('home.title')}</h1>;
}
```

## Keys

Ensure keys follow a nested structure (e.g., `home.title`, `footer.rights`) for better organization.
